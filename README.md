## PHP app to count the number of subscribers of a feed in Feedly

FeedlyのRSS登録者数を調べるツールです。

デザインはBootstrap3、Font Awesome Icons、Odometer、Slickを使っています。


### Usage

  - upload all files to the web server


### Features

  - written in PHP
